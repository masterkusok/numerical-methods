# Метод прогонки
Метод для решения систем уравнений, которые представляются в виде
трехдиагональной матрицы
a[] - коэффициенты на поддиагонали
b[] - коэф-ты на гл. диагонали
c[] - коэф-ты на наддиагонали

# Прямой ход
сначала как в методе гаусса спускаемся по нашим массивам коэффициентов, и
выражаем из каждой н-ой строки н-ый x,
везде будет получаться выражение вида: `xi = dp[i] - cp[i] * x(i-1)`,
в последнем уравнении уйдёт коэффициент цп, и у нас получется равенство

# 3 Обратный ход
После этого мы можем разрешить зависимости в получившихся ранее уравнениях, идём
снизу вверх:
xi = dp[i] - c[i] * x[i+1]

Определитель матрицы считается как произведение полученных деноминаторов, ну
типа, коэффициентов которые получаются у текущего икса.
