# Решение системы уравнений методом Гаусса. Поиск определителя и обратной матрицы

# Шаг 1.
Составляем расширенную матрицу

# Шаг 2. Прямой ход
Нужно привести матрицу к верхнетреугольному виду. Для этого:
- берём a11
- делим все коэф-ты первой строки на a11, чтобы получить a11 == 1
- из каждой последующей строки вычитаем первую строку умноженную на ai1, чтобы
  обнулить первый коэф-т

Таким образом мы получим 
x1 + a12\*x2 + a13\*x3 + ... + a1n\*xn = b1
0 \* x1 + x2 + ...

Потом аналогично делаем с каждой последующей строкой, только из предыдущей не
вычитаем ничего и получим верхнетреугольную матрицу.

# Шаг 3. Обратный ход
Начинаем считать значения X

для xn: `xn = b^(n-1)n / a^(n-1)nn`
и поднимаемся вверх: `x(n-1) = b^(n-1)(n-1) - a^(n-1)(n-1)n * xn`

и так короче получаем все значения X

# Шаг 4. Определитель
Определитель тут считается как произведение диагональных элементов полученных в
процессе прямого хода

# Шаг 5. Обратная матрица
Делаем +- тоже самое, только справа вместо столбца вставляем единичную матрицу
нужного размера, и во время обратного хода мы обнуляем ещё и элементы над
диагональю, чтобы получилась диагональная матрица а не врехнетреугольная. После
этого справа получится искомая обратная матрица.
