# Метод простых итераций
Метод простых итераций вообще супер симпл
каждую строку представляем в виде:
`xi = beta - sum([alphai/ai for i in row])`
ну типа по классике перенесли элементы в правую часть, поделили на коэффициент,
получили такого вида историю

xi = beta - alpha * x (x is a vector)
начальное приближение берём - xi = beta, далее просто пока не будет достигнута
нужная точность или мы не перейдём за лимит итераций подставляем новые значения
и приближаем иксы.

Есть ещё нюанс, для того чтобы эта вся история посчиталась, для начала, мы
должны переставить значения в исходной матрице (перествить строки) так, чтобы
каждый a[i][i] был типа максимальным в своей строке. Для этого мы для каждого
индекса выбираем лучшую не использованную строку и вставляем на нужный индекс.
вся идея.
